# Setup

## Environment

Create a `.env` file in the project root or `app/` folder with these values:

```env
DATABASE_URL=sqlite:///./test.db
TWILIO_ACCOUNT_SID=<your_twilio_account_sid>
TWILIO_AUTH_TOKEN=<your_twilio_auth_token>
GEMINI_API_KEY=<your_google_gemini_api_key>
```

## Install Dependencies

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r .\app\requirements.txt
```

## Run the API

```powershell
uvicorn app.main:app --reload
```

## Database Initialization

The SQLAlchemy models are defined in `app/schema/db_schema.py`.
The project does not automatically call `Base.metadata.create_all(engine)` on startup, so initialize the database manually if needed.

Example initialization script:

```python
from app.schema.db_schema import Base, engine
Base.metadata.create_all(engine)
```

## Important Files

- `app/main.py` — application entrypoint
- `app/dependency/db.py` — database connection provider
- `app/schema/db_schema.py` — SQLAlchemy database models
- `app/requirements.txt` — runtime dependency list

## Notes

- `route_embeddings.pt` is generated by `SemanticRouter` if missing.
- Twilio WhatsApp integration requires message sending permissions.
